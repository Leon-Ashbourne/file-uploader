# File uploader
A simple degraded version of cloud storage. Sign up to store all your files .

## Prerequisites
The following instructions assume that the user has already installed **Node.js (v22 or higher)** and **npm (v11 or higher)**. We assume you use supabase platform for files.
You are advised to include your own **.env** file configured appropriately for your local environment.

## How to use
1. Install the dependencies by running `npm install`.
2. Then, run `npx prisma generate` to generate a prisma client.
2. Now, run `node app.js` to build the application.

Congrats, you are all set to run the application.
